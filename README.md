# Ambitious Realism Landing Page

A modern SaaS-style landing page for an AI consultancy and creative studio built with Next.js 14+, TypeScript, and Tailwind CSS.

## Features

- Next.js 14+ with App Router
- TypeScript support
- Tailwind CSS with custom dark theme
- Glassmorphism effects
- Responsive design
- Custom color palette (deep charcoal, electric teal, neon purple, amber)

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
npm run start
```

## Technology Stack

- Next.js 14+
- TypeScript
- Tailwind CSS
- App Router
- Inter/Geist fonts
- Dark mode first design
